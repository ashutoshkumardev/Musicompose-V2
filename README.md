## Prerequisites

- AGP 7.2.1
- Min SDK 24
- Target SDK 33
- Java 11
- Kotlin 1.6.21
- Compose 1.2.0-rc02
- Android Studio Chipmunk 2021.2.0.0 or Electric Eel 2022.1.0.0
